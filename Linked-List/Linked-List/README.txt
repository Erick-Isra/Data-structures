Para compilar el archivo ListaLigadaSimple.java, Prueba.java y Coleccion.java:
1. Desde la terminal ubique el repositorio donde se encuentran los archivos.
2. Una vez en la carpeta de los archivos ejecute: javac *.java en la terminal.
3. Una vez que el proceso de compilacion se haya completado, ejecute: 'java Prueba' en la terminal para su mostrar la implementacion del codigo.
